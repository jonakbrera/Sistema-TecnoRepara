# Sistema TecnoRepara

Sistema integral de gestión para taller de reparación tecnológica.

## Tecnologías oficiales

- Java JDK 21
- JavaFX
- Apache Maven
- PostgreSQL
- Visual Studio Code
- Git y GitHub

## Módulos oficiales

1. Gestión de Usuarios
2. Registro de Clientes
3. Gestión de Productos
4. Gestión de Inventario
5. Ventas
6. Compras
7. Reportes
8. Autenticación/Login
9. Menú Principal

## Requisitos para trabajar

Cada integrante debe instalar:

- Java JDK 21
- Apache Maven
- PostgreSQL
- pgAdmin
- Visual Studio Code
- Git
